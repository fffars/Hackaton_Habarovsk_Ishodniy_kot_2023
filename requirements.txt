Для запуска приложения необходимо, чтобы рядом с запускаемым файлом была папка ".venv".
Её можно скачать по ссылке https://disk.yandex.ru/d/pTRK0lmVxJ9xlw
Или вручную создать следуя инструкциям https://docs.streamlit.io/library/get-started/installation
Необходмые библиотеки для установки в виртуальное окружение (.venv):
pandas
sklearn
catboost
scikeras
numpy

Окружение можно создать выполнив строку:
python -m venv .venv

Нужно установить Streamlit:
pip install streamlit

Активировать окружение выполнив файл ".venv\Scripts\activate.bat".
Так же там есть файл "deactivate.bat" для деактивации окружения.

После активации можно устанавливать необходимые библиотеки с помощью команд:
pip install <имя_библиотеки>

Ну или просто скачать папку по ссылке выше... В итоге эта папка используется как виртуальное окружения для запуска скрипта на Python.
Вот и всё. Развлекайтесь!
